# 📝 Todo List Java

Um gerenciador de tarefas simples feito em Java, desenvolvido para praticar conceitos de programação orientada a objetos.

## 🎯 Funcionalidades
- ➕ Adicionar novas tarefas
- 📋 Listar todas as tarefas
- ✔️ Marcar tarefas como concluídas
- ❌ Excluir tarefas
- 💾 Persistência de dados em arquivo texto

## 🛠 Tecnologias
- **Linguagem:** Java
- **Persistência:** Arquivo local (.txt)

## 📦 Como executar

```bash
# Compilar
javac src/*.java

# Executar
java -cp src Main
```

## 📁 Estrutura
```
src/
├── Main.java       # Menu CLI
├── Task.java       # Classe modelo de tarefa
└── TaskManager.java # Gerenciador de tarefas
```
